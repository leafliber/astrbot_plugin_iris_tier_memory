# Iris Tier Memory

AstrBot 分层记忆系统插件

## 简介

Iris Tier Memory 是一个为 AstrBot 设计的分层记忆系统插件，提供智能的记忆管理功能。

## 功能特性

- 分层记忆架构
- 智能记忆检索
- 模块化设计

## 安装

将此插件放入 AstrBot 的 plugins 目录下即可。

## 开发

### 项目结构

```
astrbot_plugin_iris_tier_memory/
├── main.py          # 入口文件
├── iris_memory/     # 业务模块
└── tests/           # 测试代码
```
